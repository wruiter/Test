Save your customer details efficiently!!

We all know, by default Odoo lets the customers enter only one billing address on the website. There is no provision to save more than one billing address. They can add a different shipping address but only at the time of check out.

So, next time they can select the one they want with just one click and add them to their order.

Why do we need an address book?
  • Provides the simplest way to the website customers to manage their address
  • Manages all the address in customer address book
  • Add a new address to the address book
  • Customer can update any existing address through address book
  • Customer can delete any existing address through address book
  • Manage billing address on the checkout page
  • Selection of billing addresses at the checkout page
  • Customer can also update any existing address during checkout
  • Customer can also delete any existing address during checkout
